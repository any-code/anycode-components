<iconic-button/>

iconic-button is a keyboard aware visual button. It contains a single icon, sized in root ems. iconic-button can be used with an icon font symbol or just plain text providing a consistent look and feel to icon and button UI elements

Attributes

By holding the shift key all iconic-button's with hotkeys assigned will transform to show their hotkey

Attribute Name Purpose Value Default Required
name A unique name by which a containing riot-tag can reference the iconic-button
this.tags.butonName		 buttonName not named
float Control the layout for different containers by assigning a float instruction left | right | none none
size Control the size of the iconic-button. 1..10 none *
hotkey a keyboard character that can be used to trigger the buttons onclick event a..z, 0..9 none

Code Sample

iconic-button {
    margin-right: 4rem;
    margin-bottom: 2rem;
}
<app>
    <iconic-button name="buttonName" float="left" size="10" color="#0A9" onclick="{ opts.alert }">+</iconic-button>
    <iconic-button name="buttonName" float="left" size="1" color="#0A9" onclick="{ opts.alert }">+</iconic-button>
</app>
riot.mount('app', {
    alert: function(e) {
        alert(e.target.outerHTML);
    }
})


<iconic-navigation/>

iconic-navigation is a vertical navigation list composed of iconic naviagtion items

Sample

  • y Yes
  • n No

Content

A standard ul element is provided as content

Attributes

Attribute Name Purpose Value Default Required
data-fixed specify if iconic-navigation should be fixed to the left or right of the viewport left | right | none none

Code Sample

<app>
    <iconic-navigation data-fixed="left">
        <ul>
            <li><a href="#iconic-button"><iconic-button float="left" size="3.2" inset color="#0B9" hotkey="a">a</iconic-button> iconic-button</a></li>
            <li><a href="#iconic-navigation"><iconic-button float="left" size="3.2" inset color="#0B9" hotkey="b">b</iconic-button> iconic-navigation</a></li>
        </ul>
    </iconic-navigation>
</app>
riot.mount('app')
